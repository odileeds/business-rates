Notes

Rows

The number of rows we found in the file. If you have empty or partial rows in your file, it will likely mess up other checks.

Valid required headings

This tells you how many of the required column headings match the format specification precisely. Headings with uppercase characters in the wrong place, extra spaces, or extra content won't count. This is used to calculate the score.

Includes required columns?

In the Business Rates format definition there are 10 columns that are listed as required. We've tried to identify how many exist by checking the column headings with a looser matching than the "Valid required headings" check. If your column headings are too different, we won't be able to find them. This is used to calculate the score.

Includes empties?

Does the file include a column Occupied that lists the empties? This is needed if we want to encourage empty properties to become occupied. This is used to calculate the score.

Valid coordinates?

Does the file contain Latitude and Longitude columns and do the cells contain things that look like coordinates? These are necessary if we want to put the properties on a map. If you are a Local Authority and only have postcodes, you can use the ONS Postcode Lookup to get latitudes and longitudes. We've even created a tool that will add latitude/longitude to CSV files with a Postcode column. This is used to calculate the score.

Valid dates

Dealing with dates is always a headache because there are so many different formats. The Government Digital Service recommend using ISO8601 which is an international standard. Dates should be YYYY-MM-DD. We've created the CSV Cleaner tool to help tidy dates into ISO8601 for you. Just drop your CSV file into it and click "Save". This is used to calculate the score.

Valid currency values

These should be given in GBP but just as a decimal number. Don't include commas or currency symbols which can be difficult to deal with consistently. This is used to calculate the score.

Is the file hosted by the Local Authority?

Ideally, files will be hosted and maintained by a Local Authority either on their own website or on a Data Store such as Data Mill North, Calderdale Data Works, Northern Data Hub, or York Open Data. That way the latest version of the Business Rates will be available. This is used to calculate the score.

Is CORS enabled?

If the file is hosted remotely, CORS will need to be enabled so that the file can be loaded by the webpage. This is used to calculate the score.
